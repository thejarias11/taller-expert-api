🎯 CÓMO USAR LA NUEVA FUNCIONALIDAD
===============================================================================

PARA ACCEDER A GESTIÓN DE USUARIOS:
1. Iniciar sesión como admin/super_admin
2. Ir a la pestaña "👥 Usuarios"
3. La sección aparecerá automáticamente si tienes permisos

PARA CREAR UN NUEVO USUARIO:
1. En la sección Usuarios, hacer clic en "Cargar Usuarios"
2. Hacer clic en "+ Crear Usuario"
3. Llenar el formulario:
   - Nombre completo
   - Username (único)
   - Email (opcional)
   - Contraseña (mín. 6 caracteres)
   - Confirmar contraseña
   - Seleccionar rol
4. Hacer clic en "Crear Usuario"

PARA ELIMINAR UN USUARIO:
1. En la lista de usuarios
2. Hacer clic en "🗑️ Eliminar" junto al usuario
3. Confirmar la eliminación
4. El usuario se desactiva (no se elimina físicamente)

PARA VER USUARIOS EXISTENTES:
1. Hacer clic en "🔄 Cargar Usuarios"
2. Se muestra la lista con:
   - Nombre completo
   - Username
   - Email
   - Rol con colores diferenciados
   - Fecha de creación
   - Acciones disponibles

🔧 ENDPOINTS API DISPONIBLES
===============================================================================

POST /api/auth/register
- Crear nuevo usuario
- Requiere: admin/super_admin
- Body: {nombre, username, email, password, role}

GET /api/auth/users  
- Obtener lista de usuarios
- Requiere: admin/super_admin
- Respuesta: Array de usuarios (sin contraseñas)

GET /api/auth/users/:id
- Obtener usuario específico
- Requiere: admin/super_admin
- Respuesta: Datos del usuario

DELETE /api/auth/users/:id
- Desactivar usuario
- Requiere: admin/super_admin
- Protege al admin principal
